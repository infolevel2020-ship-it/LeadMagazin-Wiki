## 📌 ОТЛОЖЕННЫЕ ЗАДАЧИ (вернуться позже)

**Инфраструктура / хостинг:**
- Почистить старые локальные бэкапы на хостинге: папка `bitrix/backup/`, 20 файлов от 10.12.2025, ~1.77 ГБ
- Перевести агентов с хитов на cron
- Удалить копию из облака Битрикса (убрать красную полосу) — хостинг-бэкап уже страхует
- Резервное копирование — РЕШЕНО через reg.ru (ежедневное авто)
- Настроить auto_prepend_file в php.ini для полной защиты веб-антивируса (опционально)

**Безопасность (некритичные, делать осторожно / через поддержку reg.ru):**
- PHAR расширение (php.ini — может задеть модули)
- URL wrappers — чтение файлов по URL (php.ini)
- Порядок формирования _REQUEST (php.ini)
- secure для сессионной cookie
- Automatic MIME Type Detection
- Временные файлы в корне проекта (CTempFile)
- Ограничение хостов/доменов ⚠️ (можно потерять доступ при ошибке)
- Защита административной части секретным ключом (когда сайт боевой — нужно запомнить ключ)
- Контроль целостности файлов (нужен мастер-пароль, перезапуск после обновлений)
- Двухэтапная авторизация / одноразовые пароли (для себя)
- Проверить что display_errors=Off подхватился сканером

**Пользователи (когда появятся реальные продавцы/покупатели):**
- История входов с устройств + оповещения о входе с нового устройства
