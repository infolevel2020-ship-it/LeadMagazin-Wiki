## Enterprise фундамент — что закладываем сейчас [ДжПТ + Клод]

### ✅ Закладываем сейчас:
- Чистую структуру URL
- Нормальную модель данных
- Разделение контента и бизнес-логики
- Возможность вынести биржу на отдельный сервис
- API-first мышление (даже если API пока нет)

### ❌ НЕ закладываем сейчас:
- Микросервисы
- Kubernetes
- Сложную DevOps-инфраструктуру
- Десятки серверов
- Преждевременную оптимизацию

## Структура шаблона Битрикс [Клод]
```
/local/templates/         ← дочерний шаблон (вся наша работа здесь)
/local/components/        ← кастомные компоненты
/local/modules/           ← кастомные модули
/bitrix/templates/aspro_corp3/  ← оригинал Аспро (не трогаем)
```

## Структура URL (проектируется) [Клод]
```
/                         ← главная
/catalog/                 ← каталог лидов
/catalog/stroitelstvo/    ← категория: строительство
/catalog/remont/          ← категория: ремонт
/sell/                    ← продать лиды
/buy/                     ← купить лиды
/cabinet/                 ← личный кабинет
/blog/                    ← блог
/blog/category/           ← категории блога
```
