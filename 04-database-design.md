## Ключевые сущности [ДжПТ]

### Лид (Lead)
- ID
- Категория
- Регион
- Описание
- Контактные данные
- Статус (новый / продан / возврат)
- Цена
- Продавец (кто разместил)
- Дата создания

### Пользователь (User)
- ID
- Роль (покупатель / продавец / admin)
- Имя, контакты
- Баланс
- Рейтинг
- Статус верификации [Клод]
- Дата регистрации

### Категория (Category)
- ID
- Название
- Slug (для URL)
- Описание
- Родительская категория

### Регион (Region)
- ID
- Название
- Slug

### Сделка (Deal)
- ID
- Лид
- Продавец
- Покупатель
- Сумма
- Статус (открыта / завершена / спор)
- Дата

### Платёж (Payment)
- ID
- Пользователь
- Сумма
- Тип (пополнение / списание / возврат)
- Статус
- Дата

### Баланс (Balance)
- ID
- Пользователь
- Сумма
- Последнее обновление

## Бизнес-процессы [ДжПТ]

1. **Размещение лида** — продавец создаёт лид, указывает категорию, регион, цену
2. **Модерация** — администратор проверяет лид перед публикацией
3. **Покупка** — покупатель оплачивает лид, деньги списываются с баланса
4. **Возврат** — покупатель оспаривает качество лида
5. **Рейтинг** — оценка продавца после сделки
6. **Арбитраж** — администратор разбирает спорные ситуации

## Дополнительные процессы [Клод]

7. **Верификация продавцов** — проверка перед допуском к продаже
8. **Антифрод** — защита от фейковых и дублирующихся лидов
9. **Система рейтингов и отзывов** — репутация участников
