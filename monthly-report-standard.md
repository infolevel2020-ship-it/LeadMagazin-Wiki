# Стандарт ежемесячного отчёта LeadMagazin

## Цель

Когда загружается месячный Excel-файл, нужно посчитать два отдельных блока:

1. **Оплаты клиентов**
2. **Мои продажи**

По каждому блоку нужно вывести:

* сколько денег на ГО;
* сколько денег на ИП Лесников;
* сколько по каждому продукту;
* общий итог.

---

# Основные столбцы

## Столбец A

Фактическая сумма денег.

Это главный столбец для расчёта выручки.

---

## Столбец B

Количество начисленных номеров.

Это не деньги.

---

## Столбец C

Описание начисления:

* кабинет;
* продукт;
* сумма;
* количество номеров;
* комментарий.

---

## Столбец D

Описание продажи:

* менеджер;
* плательщик;
* юридическое лицо;
* договор;
* продукт;
* комментарий.

---

## Столбец E

Особые отметки.

---

# Юридические лица

Есть два основных направления:

1. **ГО**
2. **ИП Лесников**

Важно:

**ИП Лесников не обязательно находится отдельным блоком внизу файла.**

Строки ИП Лесников могут быть разбросаны по всему файлу.

---

# Как определять ГО / ИП Лесников

## ИП Лесников

Строка относится к **ИП Лесников**, если в строке есть красная пометка:

* Договор на ИП Лесникова
* Договор на меня
* ИП Лесников
* аналогичная красная пометка, означающая договор на ИП Лесникова

## ГО

Если такой красной пометки нет, строка относится к **ГО**.

---

# Продукты

Продукт определяется по цвету строки.

| Цвет строки               | Продукт    |
| ------------------------- | ---------- |
| Песочный / бежевый        | ГЦК        |
| Голубой                   | Технология |
| Тёмно-синий               | Тест       |
| Белый                     | КЦ         |
| Коричневый любого оттенка | ПЛ         |

## Расшифровки

* **ГЦК** — ГЦК
* **Технология** — технология
* **Тест** — тест
* **КЦ** — колл-центр
* **ПЛ** — подтверждённые лиды

---

# Важное правило

Нельзя определять продукт только по тексту.

Главное правило:

**Продукт определяется по цвету строки.**

Текст в столбцах C и D используется как дополнительная проверка.

Пример:

Если строка голубая и сумма 400 000 ₽, то это **Технология**, даже если в тексте написано «Тариф Макси».

---

# Блок 1. Оплаты клиентов

Для оплат клиентов нужно посчитать таблицу:

| Продукт    | ГО | ИП Лесников | Всего |
| ---------- | -: | ----------: | ----: |
| ГЦК        |    |             |       |
| КЦ         |    |             |       |
| ПЛ         |    |             |       |
| Технология |    |             |       |
| Тест       |    |             |       |
| **ИТОГО**  |    |             |       |

---

# Блок 2. Мои продажи

Для таблицы «Мои продажи» действует то же правило:

* продукт определяется по цвету строки;
* юрлицо определяется по красной пометке;
* красная пометка = ИП Лесников;
* без красной пометки = ГО.

Нужно вывести таблицу:

| Продукт    | ГО | ИП Лесников | Всего |
| ---------- | -: | ----------: | ----: |
| ГЦК        |    |             |       |
| КЦ         |    |             |       |
| ПЛ         |    |             |       |
| Технология |    |             |       |
| Тест       |    |             |       |
| **ИТОГО**  |    |             |       |

---

# Проверки

После расчёта обязательно проверить:

1. Итог ГО + ИП Лесников = общий итог.
2. Итог по продуктам = общий итог.
3. В таблице «Мои продажи» красные строки не относить к ГО.
4. Голубые строки считать как Технология.
5. Песочные строки считать как ГЦК.
6. Белые строки считать как КЦ.
7. Коричневые строки считать как ПЛ.
8. Тёмно-синие строки считать как Тест.

---

# Команда для запуска отчёта

Когда пользователь загрузит файл за следующий месяц, команда:

**Посчитай месячный отчёт LeadMagazin по стандарту**

После этой команды нужно сразу выдать:

1. Оплаты клиентов.
2. Мои продажи.
3. Общий итог по продуктам.
4. При необходимости — ошибки и расхождения.
