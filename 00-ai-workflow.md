
# Работа с ИИ

Для работы над проектом ЛидМагазин используются два ИИ-помощника:

* ChatGPT (OpenAI)
* Claude (Anthropic)

Оба ИИ участвуют в проектировании, анализе, архитектуре, SEO, бизнес-процессах, разработке документации и подготовке технических заданий.

При наличии разных мнений:

1. Получить рекомендации от ChatGPT.
2. Получить рекомендации от Claude.
3. Сравнить результаты.
4. Зафиксировать итоговое решение в документации проекта.

Главным источником правды являются файлы проекта в репозитории GitHub, а не отдельные чаты.

---

# Работа с ИИ

Для ревью документа:

1. Открыть нужный файл.
2. Скопировать его содержимое в ChatGPT.
3. Написать:

"Проведи ревью и обнови документ.
Учитывай предыдущие архитектурные решения проекта ЛидМагазин."

4. Полученную версию проверить и сохранить.
5. При необходимости отправить тот же документ в Claude для второго мнения.

## Роли ИИ в проекте

Для разработки проекта ЛидМагазин используются два ИИ-инструмента:

### Claude Code

Используется как основной помощник по работе с проектом и кодовой базой.

Задачи:

* чтение всего проекта;
* работа с репозиторием;
* работа с файлами и документацией;
* редактирование документов;
* написание и рефакторинг кода;
* работа внутри VS Code;
* сопровождение структуры проекта.

### ChatGPT

Используется как внешний архитектор и аналитик проекта.

Задачи:

* архитектура проекта;
* бизнес-модель;
* SEO-стратегия;
* структура маркетплейса;
* проектирование базы данных;
* анализ и проверка решений;
* аудит документации;
* второе мнение по архитектурным вопросам;
* помощь в планировании развития проекта.

### Важный принцип

Claude Code и ChatGPT не конкурируют между собой.

Они используются для разных задач и дополняют друг друга.

Claude Code отвечает за работу внутри проекта и репозитория.

ChatGPT используется для стратегических, архитектурных и аналитических задач.

Все ключевые решения проекта должны фиксироваться в документации LeadMagazin-Wiki.
