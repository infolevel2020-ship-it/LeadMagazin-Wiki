## Структура URL [ФИНАЛЬНОЕ РЕШЕНИЕ]
Категории лидов — в КОРНЕ сайта (URL = поисковый запрос): `/lidy-na-nedvizhimost/`, `/lidy-na-bankrotstvo/` и т.д.
`/catalog/` — хаб каталога/биржи лидов, НЕ родитель категорий.
Подробности и реестр слагов — в файле `08-semantic-core.md`.

## СТАНДАРТ распределения ключей по странице категории [Клод + Я]

Одна страница категории закрывает НЕСКОЛЬКО близких запросов БЕЗ дублей. Ключи распределяются по элементам:

| Элемент | Что ставим | Какой запрос ловит |
|---------|-----------|--------------------|
| URL | `/lidy-na-nedvizhimost/` | "лиды на/для недвижимости" |
| H1 | Лиды на недвижимость | "лиды на недвижимость" |
| Хлебные крошки | Лиды на недвижимость | навигация |
| Title | **Купить** лиды на недвижимость — заявки для риэлторов и агентств \| ЛидМагазин | "купить лиды на недвижимость" |
| Description | Купить готовые лиды и заявки на недвижимость... целевые клиенты | "заявки", "клиенты" |
| Текст H2/абзацы | естественно вписать "купить заявки на недвижимость", "клиенты для агентства" | синонимичные запросы |

**Принцип:** URL и H1 — основной запрос ("лиды на X"). Title — коммерческий усилитель ("купить лиды на X"). Description и текст — синонимы (заявки, клиенты). Так одна страница ранжируется по всему кластеру запросов, не создавая дублей (см. решение против каннибализации в файле 08).

Этот шаблон применяется ко ВСЕМ категориям единообразно.

## ТРЁХУРОВНЕВАЯ СТРУКТУРА КАТАЛОГА [Я + Клод]

Каталог имеет 3 уровня, и ВСЕ ТРИ — посадочные, заточенные под захват заявки (главная цель сайта).

```
Уровень 1 — КАТЕГОРИЯ (посадочная ниши)
/lidy-na-nedvizhimost/
H1: Лиды на недвижимость

Уровень 2 — ПОДКАТЕГОРИЯ (посадочная сегмента)
/lidy-na-nedvizhimost/novostroy/
H1: Лиды на новостройки

Уровень 3 — КАРТОЧКА (максимально посадочная, технически = карточка товара Аспро)
/lidy-na-nedvizhimost/novostroy/kvartiry-moskva-zhk-molodezhnyy/
H1: Квартиры в Москве, ЖК Молодёжный
```

| Уровень | Роль | Главная задача |
|---------|------|----------------|
| Категория | посадочная ниши | широкий запрос → заявка или переход вглубь |
| Подкатегория | посадочная сегмента | уточнённый запрос → заявка |
| Карточка | макс. посадочная | конкретика → ЗАХВАТ КОНТАКТА |

⚠️ Технический момент: проверить что Аспро поддерживает двухуровневые разделы (категория + подкатегория) — при технической настройке URL.

## ШАБЛОН ПРОДАЮЩЕЙ КАРТОЧКИ (Уровень 3) [Я + Клод]

Карточка — ГЛАВНАЯ точка конверсии. Технически это карточка товара Аспро, но по сути — продающий лендинг. Задача: объяснить → предложить → продать → ВЗЯТЬ КОНТАКТ, чтобы позвонил МПП и подобрал вариант под задачи клиента.

**Блоки карточки:**
1. H1 + краткая суть предложения
2. Что это за лиды / что входит
3. Выгода для клиента (зачем брать, что получит)
4. Кнопки CTA: **Заказать** / **Задать вопрос** (заметные, повторяются по странице)
5. Как работает (получите контакт → звонок МПП → подбор под задачу)
6. Доверие (гарантии, примеры, отзывы)
7. Форма захвата контакта
8. FAQ

**Акцент:** максимальное ударение на захват контакта. Не продажа онлайн, а заявка → звонок менеджера.
